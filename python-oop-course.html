Understanding Classes and Objects

What is Object-Oriented Programming?

Object-Oriented Programming (OOP) is a programming paradigm that organizes code into objects that contain both data and code. It's a powerful way to model real-world entities and relationships.

Core OOP Concepts:

  • Classes: Blueprints for creating objects
  • Objects: Instances of classes with specific data
  • Attributes: Data stored in objects (variables)
  • Methods: Functions that operate on object data
  • Encapsulation: Bundling data and methods together
  • Inheritance: Creating new classes from existing ones
  • Polymorphism: Same interface, different implementations

Benefits of OOP:

  • Organization: Code is organized into logical units
  • Reusability: Classes can be reused in different programs
  • Maintainability: Easier to modify and extend code
  • Modeling: Natural way to model real-world entities
  • Abstraction: Hide complex implementation details

Real-World Analogy:

  • Class: Like a car blueprint (design)
  • Object: Like an actual car (instance)
  • Attributes: Like car properties (color, model, year)
  • Methods: Like car actions (start, stop, accelerate)
Basic Class and Object Example
# Defining a simple class
class Car:
    """A simple class to represent a car."""
    
    def __init__(self, brand, model, year):
        """Initialize car attributes."""
        self.brand = brand
        self.model = model
        self.year = year
        self.is_running = False
    
    def start_engine(self):
        """Start the car engine."""
        if not self.is_running:
            self.is_running = True
            return f"{self.brand} {self.model} engine started!"
        else:
            return "Engine is already running!"
    
    def stop_engine(self):
        """Stop the car engine."""
        if self.is_running:
            self.is_running = False
            return f"{self.brand} {self.model} engine stopped!"
        else:
            return "Engine is already stopped!"
    
    def get_info(self):
        """Get car information."""
        status = "running" if self.is_running else "stopped"
        return f"{self.year} {self.brand} {self.model} - Engine: {status}"

# Creating objects (instances) of the Car class
my_car = Car("Toyota", "Camry", 2020)
your_car = Car("Honda", "Civic", 2019)

# Using object methods
print(my_car.start_engine())  # Toyota Camry engine started!
print(my_car.get_info())      # 2020 Toyota Camry - Engine: running
print(my_car.stop_engine())   # Toyota Camry engine stopped!

print(your_car.get_info())    # 2019 Honda Civic - Engine: stopped

# Accessing object attributes
print(f"Brand: {my_car.brand}")
print(f"Model: {my_car.model}")
print(f"Year: {my_car.year}")

# Multiple objects of the same class
cars = [
    Car("Ford", "Mustang", 2021),
    Car("BMW", "X5", 2022),
    Car("Tesla", "Model 3", 2023)
]

for car in cars:
    print(car.get_info())

Defining Your Own Data Types

Class Definition Syntax

Classes are defined using the `class` keyword followed by the class name. They serve as templates for creating objects with specific attributes and behaviors.

Class Structure:

  • class keyword: Used to define a new class
  • Class name: Use PascalCase (e.g., UserAccount)
  • Docstring: Documentation for the class
  • Attributes: Variables that store data
  • Methods: Functions that operate on the data
  • Constructor: Special method called `__init__`

The `__init__` Method:

  • Constructor: Called when creating a new object
  • Self parameter: Refers to the object being created
  • Initialization: Set up initial attribute values
  • Parameters: Accept data to initialize the object
  • Required: Every class should have an `__init__` method

Naming Conventions:

  • Class names: PascalCase (UserAccount, Car, Bank)
  • Method names: snake_case (get_balance, set_name)
  • Attribute names: snake_case (user_name, account_balance)
  • Private attributes: Leading underscore (_private_var)
  • Constants: UPPER_CASE (MAX_BALANCE)
Class Creation Examples
# Basic class definition
class BankAccount:
    """A simple bank account class."""
    
    def __init__(self, account_holder, initial_balance=0):
        """Initialize a new bank account."""
        self.account_holder = account_holder
        self.balance = initial_balance
        self.account_number = self._generate_account_number()
    
    def _generate_account_number(self):
        """Generate a random account number."""
        import random
        return f"ACC{random.randint(10000, 99999)}"
    
    def deposit(self, amount):
        """Deposit money into the account."""
        if amount > 0:
            self.balance += amount
            return f"Deposited ${amount}. New balance: ${self.balance}"
        else:
            return "Invalid deposit amount"
    
    def withdraw(self, amount):
        """Withdraw money from the account."""
        if amount > 0 and amount <= self.balance:
            self.balance -= amount
            return f"Withdrew ${amount}. New balance: ${self.balance}"
        else:
            return "Invalid withdrawal amount or insufficient funds"
    
    def get_balance(self):
        """Get current account balance."""
        return f"Balance: ${self.balance}"
    
    def get_account_info(self):
        """Get account information."""
        return f"Account: {self.account_number}, Holder: {self.account_holder}"

# Creating objects
account1 = BankAccount("Alice Johnson", 1000)
account2 = BankAccount("Bob Smith")

# Using methods
print(account1.deposit(500))      # Deposited $500. New balance: $1500
print(account1.withdraw(200))     # Withdrew $200. New balance: $1300
print(account1.get_balance())     # Balance: $1300
print(account1.get_account_info()) # Account: ACC12345, Holder: Alice Johnson

# Class with default values
class Student:
    """A student class with default values."""
    
    def __init__(self, name, age=18, grade_level="Freshman"):
        self.name = name
        self.age = age
        self.grade_level = grade_level
        self.courses = []
    
    def add_course(self, course):
        """Add a course to student's schedule."""
        if course not in self.courses:
            self.courses.append(course)
            return f"Added {course} to {self.name}'s schedule"
        else:
            return f"{self.name} is already enrolled in {course}"
    
    def get_schedule(self):
        """Get student's course schedule."""
        if self.courses:
            return f"{self.name}'s courses: {', '.join(self.courses)}"
        else:
            return f"{self.name} has no courses enrolled"

# Creating students with different parameters
student1 = Student("Alice", 20, "Junior")
student2 = Student("Bob")  # Uses default values

print(student1.add_course("Python Programming"))
print(student1.add_course("Data Structures"))
print(student1.get_schedule())

# Complete Library Management System
class Book:
    def __init__(self, title, author, isbn):
        self.title = title
        self.author = author
        self.isbn = isbn
        self.is_borrowed = False
        self.borrower = None
    
    def borrow(self, user):
        if not self.is_borrowed:
            self.is_borrowed = True
            self.borrower = user
            return f"'{self.title}' borrowed by {user.name}"
        return f"'{self.title}' is already borrowed"
    
    def return_book(self):
        if self.is_borrowed:
            borrower = self.borrower
            self.is_borrowed = False
            self.borrower = None
            return f"'{self.title}' returned by {borrower.name}"
        return f"'{self.title}' is not borrowed"

class User:
    def __init__(self, name, user_id):
        self.name = name
        self.user_id = user_id
        self.borrowed_books = []
    
    def borrow_book(self, book):
        if book.borrow(self):
            self.borrowed_books.append(book)
            return f"{self.name} borrowed '{book.title}'"
        return f"Could not borrow '{book.title}'"
    
    def return_book(self, book):
        if book in self.borrowed_books:
            result = book.return_book()
            self.borrowed_books.remove(book)
            return result
        return f"{self.name} didn't borrow '{book.title}'"

class Library:
    def __init__(self):
        self.books = []
        self.users = []
    
    def add_book(self, book):
        self.books.append(book)
        return f"Added '{book.title}' to library"
    
    def add_user(self, user):
        self.users.append(user)
        return f"Added user {user.name}"
    
    def find_book(self, title):
        for book in self.books:
            if book.title.lower() == title.lower():
                return book
        return None
    
    def get_available_books(self):
        return [book for book in self.books if not book.is_borrowed]

# Using the library system
library = Library()

# Add books
book1 = Book("Python Programming", "John Smith", "123456")
book2 = Book("Data Structures", "Jane Doe", "789012")
library.add_book(book1)
library.add_book(book2)

# Add users
user1 = User("Alice", "U001")
user2 = User("Bob", "U002")
library.add_user(user1)
library.add_user(user2)

# Borrow and return books

print(user2.borrow_book(book2))
print(user1.return_book(book1))

# Check available books
available = library.get_available_books()
print(f"Available books: {[book.title for book in available]}")
