Creating Reusable Code Blocks

What are Functions?

Functions are reusable blocks of code that perform specific tasks. They help organize code, avoid repetition, and make programs more maintainable and readable.

Function Benefits:

  • Reusability: Write once, use many times
  • Organization: Break code into logical pieces
  • Maintainability: Easier to fix and update
  • Testing: Test individual pieces of code
  • Readability: Code becomes more self-documenting
  • Modularity: Build complex programs from simple parts

Function Components:

  • def keyword: Used to define a function
  • Function name: Choose descriptive names
  • Parameters: Input values the function receives
  • Function body: Code that runs when function is called
  • Return statement: Output value the function sends back
  • Function call: How you use the function

Real-World Analogy:

  • Function: Like a recipe with ingredients and steps
  • Parameters: Like ingredients you can change
  • Return value: Like the finished dish
  • Function call: Like following the recipe
Function Examples
# Function declaration
def greet(name):
    return f"Hello, {name}!"

# Function call
message = greet("Alice")
print(message)  # Hello, Alice!

# Function with multiple parameters
def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

# Function without parameters
def say_hello():
    print("Hello, World!")

say_hello()  # Hello, World!

# Function with no return value
def print_info(name, age):
    print(f"Name: {name}")
    print(f"Age: {age}")

print_info("Bob", 30)
# Output:
# Name: Bob
# Age: 30

# Function with default parameter
def greet_with_title(name, title="Mr."):
    return f"Hello, {title} {name}!"

print(greet_with_title("Smith"))  # Hello, Mr. Smith!
print(greet_with_title("Johnson", "Dr."))  # Hello, Dr. Johnson!

# Function that performs calculations
def calculate_area(length, width):
    area = length * width
    return area

rectangle_area = calculate_area(5, 3)
print(f"Area: {rectangle_area}")  # Area: 15

# Function with multiple return values
def get_name_and_age():
    return "Alice", 25

name, age = get_name_and_age()
print(f"Name: {name}, Age: {age}")  # Name: Alice, Age: 25

# Function with conditional logic
def check_number(num):
    if num > 0:
        return "Positive"
    elif num < 0:
        return "Negative"
    else:
        return "Zero"

print(check_number(5))   # Positive
print(check_number(-3))  # Negative
print(check_number(0))   # Zero

Passing Data to Functions

Types of Parameters

Parameters allow functions to receive data and make them more flexible and reusable. Understanding different parameter types helps you write better functions.

Required Parameters:

  • Positional: Must be provided in order
  • No defaults: Function won't work without them
  • Clear purpose: Essential for function operation
  • Examples: name, age, length, width

Default Parameters:

  • Optional: Have default values
  • Can be omitted: Function uses default if not provided
  • Must come last: After required parameters
  • Mutable defaults: Be careful with lists/dicts
  • Examples: title="Mr.", greeting="Hello"

Parameter Types:

  • Positional: Passed by position
  • Keyword: Passed by name
  • Arbitrary: *args for variable number
  • Keyword arbitrary: **kwargs for named arguments
Parameter Examples
# Required parameters
def greet(name, age):
    return f"Hello {name}, you are {age} years old"

print(greet("Alice", 25))  # Hello Alice, you are 25 years old

# Default parameters
def greet_with_title(name, title="Mr.", greeting="Hello"):
    return f"{greeting} {title} {name}!"

print(greet_with_title("Smith"))                    # Hello Mr. Smith!
print(greet_with_title("Johnson", "Dr."))          # Hello Dr. Johnson!
print(greet_with_title("Brown", "Ms.", "Goodbye")) # Goodbye Ms. Brown!

# Keyword arguments
def create_profile(name, age, city, country="USA"):
    return f"Name: {name}, Age: {age}, City: {city}, Country: {country}"

# Using keyword arguments (order doesn't matter)
profile1 = create_profile(name="Alice", age=25, city="New York")
profile2 = create_profile(city="London", name="Bob", age=30, country="UK")

print(profile1)  # Name: Alice, Age: 25, City: New York, Country: USA
print(profile2)  # Name: Bob, Age: 30, City: London, Country: UK

# Arbitrary arguments (*args)
def sum_all(*numbers):
    return sum(numbers)

print(sum_all(1, 2, 3))        # 6
print(sum_all(1, 2, 3, 4, 5))  # 15

# Keyword arbitrary arguments (**kwargs)
def create_person(**attributes):
    return attributes

person = create_person(name="Alice", age=25, city="New York", job="Engineer")
print(person)  # {'name': 'Alice', 'age': 25, 'city': 'New York', 'job': 'Engineer'}

# Mixed parameter types
def complex_function(required, optional="default", *args, **kwargs):
    print(f"Required: {required}")
    print(f"Optional: {optional}")
    print(f"Args: {args}")
    print(f"Kwargs: {kwargs}")

complex_function("hello", "world", 1, 2, 3, name="Alice", age=25)
# Output:
# Required: hello
# Optional: world
# Args: (1, 2, 3)
# Kwargs: {'name': 'Alice', 'age': 25}

# Common parameter patterns
def calculate_stats(*numbers):
    """Calculate statistics for any number of values."""
    if not numbers:
        return {"error": "No numbers provided"}
    
    return {
        "sum": sum(numbers),
        "average": sum(numbers) / len(numbers),
        "min": min(numbers),
        "max": max(numbers),
        "count": len(numbers)
    }

stats = calculate_stats(1, 2, 3, 4, 5)
print(stats)  # {'sum': 15, 'average': 3.0, 'min': 1, 'max': 5, 'count': 5}

# Function with flexible parameters
def format_message(message, prefix="", suffix="", **options):
    """Format a message with optional prefix and suffix."""
    result = f"{prefix}{message}{suffix}"
    
    if options.get("uppercase"):
        result = result.upper()
    if options.get("repeat"):
        result = result * options["repeat"]
    
    return result

print(format_message("Hello", prefix=">>> ", suffix=" <<<"))
# >>> Hello <<<

print(format_message("Hello", uppercase=True, repeat=3))
# HELLOHELLOHELLO

# Function Library - Putting it all together
def math_operations():
    """Collection of mathematical functions."""
    
    def add(*numbers):
        return sum(numbers)
    
    def multiply(*numbers):
        result = 1
        for num in numbers:
            result *= num
        return result
    
    def average(*numbers):
        return sum(numbers) / len(numbers) if numbers else 0
    
    return {"add": add, "multiply": multiply, "average": average}

def string_operations():
    """Collection of string processing functions."""
    
    def reverse_string(text):
        return text[::-1]
    
    def count_words(text):
        return len(text.split())
    
    def is_palindrome(text):
        cleaned = ''.join(c.lower() for c in text if c.isalnum())
        return cleaned == cleaned[::-1]
    
    return {"reverse": reverse_string, "count_words": count_words, "is_palindrome": is_palindrome}

def list_operations():
    """Collection of list processing functions."""
    
    def remove_duplicates(lst):
        return list(dict.fromkeys(lst))
    
    def flatten_list(nested_list):
        result = []
        for item in nested_list:
            if isinstance(item, list):
                result.extend(flatten_list(item))
            else:
                result.append(item)
        return result
    
    def chunk_list(lst, size):
        return [lst[i:i+size] for i in range(0, len(lst), size)]
    
    return {"remove_duplicates": remove_duplicates, "flatten": flatten_list, "chunk": chunk_list}

# Using the function libraries
math_lib = math_operations()
string_lib = string_operations()
list_lib = list_operations()

# Test math functions
print(math_lib["add"](1, 2, 3, 4, 5))      # 15
print(math_lib["multiply"](2, 3, 4))        # 24
print(math_lib["average"](10, 20, 30))      # 20.0

# Test string functions
text = "A man a plan a canal Panama"
print(string_lib["reverse"](text))           # amanaP lanac a nalp a nam A
print(string_lib["count_words"](text))       # 7
print(string_lib["is_palindrome"](text))     # True

# Test list functions
numbers = [1, 2, 2, 3, 3, 4, 5, 5]
print(list_lib["remove_duplicates"](numbers))  # [1, 2, 3, 4, 5]

nested = [1, [2, 3], [4, [5, 6]], 7]
print(list_lib["flatten"](nested))           # [1, 2, 3, 4, 5, 6, 7]

data = [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(list_lib["chunk"](data, 3))            # [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
