Making Decisions and Repeating Actions

Welcome to Control Flow & Loops!

Control Flow Examples
# Conditional statement
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# For loop
for i in range(5):
    print(f"Count: {i}")

# While loop
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# Logical operators
name = "Alice"
age = 25
if name == "Alice" and age >= 18:
    print("Welcome, Alice!")

# Break statement
for i in range(10):
    if i == 5:
        break
    print(i)

Understanding Program Flow

Control Flow in Programming

Control flow determines the order in which statements are executed in a program. It allows your program to make decisions and repeat actions based on conditions.

Types of Control Flow:

  • Sequential: Code executes line by line
  • Conditional: Different paths based on conditions
  • Iterative: Repeating code with loops
  • Exception Handling: Managing errors

Why Control Flow Matters:

  • Makes programs interactive and dynamic
  • Allows programs to respond to different situations
  • Enables automation and repetition
  • Creates complex, real-world applications

Real-World Examples:

  • User authentication (if password correct)
  • Processing lists of data (for each item)
  • Game logic (while player is alive)
  • Menu systems (if user selects option)
Control Flow Examples
# Sequential execution
print("Step 1: Start")
print("Step 2: Process")
print("Step 3: End")

# Conditional execution
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Iterative execution
for i in range(3):
    print(f"Count: {i}")

# Combined control flow
user_input = "yes"
count = 0

while user_input == "yes" and count < 5:
    print(f"Loop iteration {count + 1}")
    count += 1
    if count >= 3:
        user_input = "no"

print("Loop finished")
