ðŸš€ Slide 2: Error Handling Basics

Managing Exceptions

What is Error Handling?

Error handling allows your program to deal with unexpected situations gracefully instead of crashing. In Python, we use try/except blocks to handle exceptions.

Why Handle Errors?

  • Prevent program crashes
  • Provide user-friendly error messages
  • Allow programs to recover from errors
  • Make programs more robust

Basic Structure:

  • try: - Code that might cause an error
  • except: - Code to handle the error
  • else: - Code that runs if no error occurs
  • finally: - Code that always runs
Error Handling Example
try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("Cannot divide by zero!")
except Exception as e:
    print(f"An error occurred: {e}")
else:
    print("No errors occurred!")
finally:
    print("This always runs!")

ðŸš€ Slide 3: Common Exceptions

Types of Errors in Python

Common Python Exceptions

Python has many built-in exception types. Understanding them helps you write better error handling code.

Built-in Exceptions:

  • ValueError: Invalid value for operation
  • TypeError: Wrong type for operation
  • IndexError: Invalid list index
  • KeyError: Invalid dictionary key
  • FileNotFoundError: File doesn't exist
  • ZeroDivisionError: Division by zero
  • NameError: Variable not defined

When They Occur:

  • ValueError: int("abc")
  • TypeError: "hello" + 5
  • IndexError: list[10] (list too short)
  • KeyError: dict["missing_key"]
Exception Examples
# ValueError
try:
    age = int("twenty")
except ValueError:
    print("Please enter a valid number")

# TypeError
try:
    result = "hello" + 5
except TypeError:
    print("Cannot add string and number")

# IndexError
try:
    numbers = [1, 2, 3]
    print(numbers[5])
except IndexError:
    print("Index out of range")

# KeyError
try:
    person = {"name": "Alice"}
    print(person["age"])
except KeyError:
    print("Key not found in dictionary")

Creating Your Own Error Types

Creating Custom Exceptions

You can create your own exception types to handle specific errors in your programs. This makes your code more organized and easier to understand.

How to Create Custom Exceptions:

  • Create a class that inherits from Exception
  • Usually just need an __init__ method
  • Can add custom attributes and methods
  • Use descriptive names ending in "Error"

Benefits:

  • More specific error handling
  • Better error messages
  • Easier to debug
  • More professional code
Custom Exception Example
class AgeError(Exception):
    def __init__(self, age, message="Invalid age"):
        self.age = age
        self.message = message
        super().__init__(self.message)

def check_age(age):
    if age < 0:
        raise AgeError(age, "Age cannot be negative")
    elif age > 150:
        raise AgeError(age, "Age seems unrealistic")
    return True

try:
    check_age(-5)
except AgeError as e:
    print(f"Error: {e.message} ({e.age})")

try:
    check_age(200)
except AgeError as e:
    print(f"Error: {e.message} ({e.age})")

ðŸš€ Slide 5: File Operations

Reading and Writing Files

Working with Files

File operations allow your programs to read data from files and save data to files. This is essential for many real-world applications.

File Modes:

  • 'r': Read (default)
  • 'w': Write (overwrites existing file)
  • 'a': Append (adds to existing file)
  • 'x': Exclusive creation (fails if file exists)
  • 'b': Binary mode
  • 't': Text mode (default)

Best Practices:

  • Always close files when done
  • Use with statement for automatic closing
  • Handle file not found errors
  • Use appropriate file modes
File Operations Example
# Writing to a file

    file.write("Hello, World!\n")
    file.write("This is a test file.\n")

# Reading from a file
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
with open("data.txt", "r") as file:
    for line in file:
        print(line.strip())

# Appending to a file

CSV Module:

  • Store simple databases
  • Data analysis and processing

Package Structure:

Virtual environments are isolated Python environments that allow you to install packages without affecting your system Python installation.

Why Use Virtual Environments?

ðŸš€ Slide 12: Dictionary Comprehensions

Creating Dictionaries Efficiently

Dictionary Comprehensions

Dictionary comprehensions are a concise way to create dictionaries from iterables, similar to list comprehensions.

Basic Syntax:

  • {key: value for item in iterable}
  • Can include conditions
  • More readable than loops
  • Efficient for data transformation

Common Patterns:

  • Transform list to dictionary
  • Filter data while creating
  • Create lookup tables
  • Count occurrences
Dictionary Comprehension Example
# Basic dictionary comprehension
numbers = [1, 2, 3, 4, 5]
squares = {num: num**2 for num in numbers}
print(squares)  # {1: 1, 2: 4, 3: 9, 4: 16, 5: 25}

# With condition
even_squares = {num: num**2 for num in numbers if num % 2 == 0}
print(even_squares)  # {2: 4, 4: 16}

# From two lists
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]
people = {name: age for name, age in zip(names, ages)}
print(people)  # {'Alice': 25, 'Bob': 30, 'Charlie': 35}

# Counting characters
text = "hello"
char_count = {char: text.count(char) for char in set(text)}
print(char_count)  # {'h': 1, 'e': 1, 'l': 2, 'o': 1}

ðŸš€ Slide 13: Set Operations

Mathematical Set Operations

Set Operations

Sets support mathematical set operations that are useful for data analysis and problem solving.

Basic Operations:

  • union() or |: Combine sets
  • intersection() or &: Common elements
  • difference() or -: Elements in first but not second
  • symmetric_difference() or ^: Elements in either but not both

Set Methods:

  • add(): Add element
  • remove(): Remove element (raises error if not found)
  • discard(): Remove element (no error if not found)
  • clear(): Remove all elements
Set Operations Example
set1 = {1, 2, 3, 4, 5}
set2 = {4, 5, 6, 7, 8}

# Union
union_set = set1 | set2
print(union_set)  # {1, 2, 3, 4, 5, 6, 7, 8}

# Intersection
intersection_set = set1 & set2
print(intersection_set)  # {4, 5}

# Difference
difference_set = set1 - set2
print(difference_set)  # {1, 2, 3}

# Symmetric difference
symmetric_diff = set1 ^ set2
print(symmetric_diff)  # {1, 2, 3, 6, 7, 8}

# Practical example
students_math = {"Alice", "Bob", "Charlie"}
students_science = {"Bob", "David", "Eve"}

both_classes = students_math & students_science
print(f"Students in both classes: {both_classes}")  # {'Bob'}

Password Generator Example
import random
import string

def generate_password(length=12, include_uppercase=True, 
                     include_numbers=True, include_symbols=True):
    characters = string.ascii_lowercase
    
    if include_uppercase:
        characters += string.ascii_uppercase
    if include_numbers:
        characters += string.digits
    if include_symbols:
        characters += "!@#$%^&*()_+-="
    
    password = ''.join(random.choice(characters) for _ in range(length))
    return password

def check_password_strength(password):
    score = 0
    if len(password) >= 8:
        score += 1
    if any(c.isupper() for c in password):
        score += 1
    if any(c.islower() for c in password):
        score += 1
    if any(c.isdigit() for c in password):
        score += 1
    if any(c in "!@#$%^&*()_+-=" for c in password):
        score += 1
    
    if score <= 2:
        return "Weak"
    elif score <= 3:
        return "Medium"
    else:
        return "Strong"

# Generate and check password
password = generate_password(16, True, True, True)
strength = check_password_strength(password)
print(f"Generated password: {password}")
print(f"Password strength: {strength}")

File Manager Example
import os
import shutil
from datetime import datetime

class FileManager:
    def __init__(self, directory="."):
        self.directory = directory
    
    def list_files(self):
        try:
            files = os.listdir(self.directory)
            for file in files:
                path = os.path.join(self.directory, file)
                size = os.path.getsize(path)
                modified = datetime.fromtimestamp(os.path.getmtime(path))
                print(f"{file:<20} {size:>8} bytes {modified.strftime('%Y-%m-%d %H:%M')}")
        except Exception as e:
            print(f"Error listing files: {e}")
    
    def copy_file(self, source, destination):
        try:
            shutil.copy2(source, destination)
            print(f"File copied: {source} -> {destination}")
        except Exception as e:
            print(f"Error copying file: {e}")
    
    def create_directory(self, name):
        try:
            os.makedirs(name, exist_ok=True)
            print(f"Directory created: {name}")
        except Exception as e:
            print(f"Error creating directory: {e}")
    
    def get_file_info(self, filename):
        try:
            path = os.path.join(self.directory, filename)
            if os.path.exists(path):
                size = os.path.getsize(path)
                modified = datetime.fromtimestamp(os.path.getmtime(path))
                return {
                    "name": filename,
                    "size": size,
                    "modified": modified
                }
        except Exception as e:
            print(f"Error getting file info: {e}")
        return None

# Usage
fm = FileManager()
fm.list_files()
fm.create_directory("backup")
fm.copy_file("data.txt", "backup/data.txt")

Data Analysis Example
import csv
from collections import defaultdict

class DataAnalyzer:
    def __init__(self):
        self.data = []
    
    def load_csv(self, filename):
        try:
            with open(filename, 'r') as file:
                reader = csv.DictReader(file)
                self.data = list(reader)
            print(f"Loaded {len(self.data)} records from {filename}")
        except Exception as e:
            print(f"Error loading file: {e}")
    
    def calculate_stats(self, column):
        try:
            values = [float(row[column]) for row in self.data if row[column]]
            if values:
                return {
                    "count": len(values),
                    "sum": sum(values),
                    "average": sum(values) / len(values),
                    "min": min(values),
                    "max": max(values)
                }
        except Exception as e:
            print(f"Error calculating stats: {e}")
        return None
    
    def group_by(self, group_column, value_column):
        groups = defaultdict(list)
        for row in self.data:
            if row[group_column] and row[value_column]:
                groups[row[group_column]].append(float(row[value_column]))
        
        result = {}
        for group, values in groups.items():
            result[group] = {
                "count": len(values),
                "average": sum(values) / len(values)
            }
        return result
    
    def filter_data(self, column, value):
        return [row for row in self.data if row.get(column) == value]

# Usage
analyzer = DataAnalyzer()
analyzer.load_csv("sales.csv")
stats = analyzer.calculate_stats("amount")
print(f"Sales statistics: {stats}")

Web Scraper Example
import requests
import re
from urllib.parse import urljoin, urlparse

class WebScraper:
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36'
        })
    
    def fetch_page(self, url):
        try:
            response = self.session.get(url)
            response.raise_for_status()
            return response.text
        except Exception as e:
            print(f"Error fetching {url}: {e}")
            return None
    
    def extract_links(self, html, base_url):
        links = re.findall(r'href=["\']([^"\']+)["\']', html)
        absolute_links = []
        for link in links:
            absolute_link = urljoin(base_url, link)
            if urlparse(absolute_link).netloc == urlparse(base_url).netloc:
                absolute_links.append(absolute_link)
        return absolute_links
    
    def extract_text(self, html):
        # Remove HTML tags
        text = re.sub(r'<[^>]+>', '', html)
        # Remove extra whitespace
        text = re.sub(r'\s+', ' ', text).strip()
        return text
    
    def save_data(self, data, filename):
        try:
            with open(filename, 'w', encoding='utf-8') as file:
                file.write(data)
            print(f"Data saved to {filename}")
        except Exception as e:
            print(f"Error saving data: {e}")

# Usage
scraper = WebScraper()
html = scraper.fetch_page("https://example.com")
if html:
    text = scraper.extract_text(html)
    scraper.save_data(text, "extracted_text.txt")

Optimization Example
import time

# Slow version
def slow_find_duplicates(numbers):
    duplicates = []
    for i in range(len(numbers)):
        for j in range(i + 1, len(numbers)):
            if numbers[i] == numbers[j] and numbers[i] not in duplicates:
                duplicates.append(numbers[i])
    return duplicates

# Fast version
def fast_find_duplicates(numbers):
    seen = set()
    duplicates = set()
    for num in numbers:
        if num in seen:
            duplicates.add(num)
        else:
            seen.add(num)
    return list(duplicates)

# Test performance
numbers = [1, 2, 3, 4, 5, 1, 2, 6, 7, 8, 9, 1] * 1000

start = time.time()
result1 = slow_find_duplicates(numbers)
end = time.time()
print(f"Slow version: {end - start:.4f} seconds")

start = time.time()
result2 = fast_find_duplicates(numbers)
end = time.time()
print(f"Fast version: {end - start:.4f} seconds")

def calculate_monthly_payment(principal, annual_rate, years):
    """
    Calculate monthly mortgage payment.
    
    Args:
        principal (float): Loan amount
        annual_rate (float): Annual interest rate (as decimal)
        years (int): Loan term in years
    
    Returns:
        float: Monthly payment amount
    """
    monthly_rate = annual_rate / 12
    num_payments = years * 12
    
    if monthly_rate == 0:
        return principal / num_payments
    
    payment = principal * (monthly_rate * (1 + monthly_rate)**num_payments) / \
              ((1 + monthly_rate)**num_payments - 1)
    return round(payment, 2)
