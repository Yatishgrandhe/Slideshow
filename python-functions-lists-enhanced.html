Functions & Lists Examples
# Function definition
def greet(name):
    return f"Hello, {name}!"

# Function call
message = greet("Alice")
print(message)

# List creation
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]

# List operations
numbers.append(6)
names.remove("Bob")

# Function with list parameter
def calculate_average(numbers):
    if len(numbers) == 0:
        return 0
    return sum(numbers) / len(numbers)

# List comprehension
squares = [x**2 for x in range(5)]

# Function returning list
def get_even_numbers(max_num):
    return [x for x in range(max_num) if x % 2 == 0]

ðŸ”§ Slide 2: What is a Function?

Reusable Blocks of Code

Understanding Functions

A function is a named block of code that performs a specific task. You can call a function whenever you need to perform that task, which helps you avoid repeating code.

  • Functions make code organized and reusable
  • They can take input (parameters) and return output (values)
  • Python has built-in functions and you can create your own
Function Example
# Defining a function
def greet():
    print("Hello!")

# Calling a function
greet()  # Output: Hello!

ðŸ”§ Slide 3: Why Use Functions?

Benefits of Functions

Why Functions Matter

  • Reusability: Write code once, use it many times
  • Organization: Break big problems into smaller pieces
  • Readability: Make code easier to understand
  • Testing: Test small parts of code independently
Reusable Function
def say_hello(name):
    print(f"Hello, {name}!")

say_hello("Alice")
say_hello("Bob")

ðŸ”§ Slide 4: Function Syntax

How to Write Functions

Defining Functions in Python

  • Use the def keyword
  • Give your function a name
  • Use parentheses for parameters (even if empty)
  • End the line with a colon :
  • Indent the function body
Syntax Example
def add(a, b):
    result = a + b
    print(result)

add(2, 3)  # Output: 5

ðŸ”§ Slide 5: Parameters and Arguments

Passing Data to Functions

Parameters vs. Arguments

  • Parameter: Variable in the function definition
  • Argument: Value you pass to the function
  • You can have multiple parameters, separated by commas
  • Arguments are matched to parameters by position or name
Parameters Example
def multiply(x, y):
    print(x * y)

multiply(4, 5)  # Output: 20

ðŸ”§ Slide 6: Default Parameters

Optional Function Arguments

Default Parameter Values

  • You can give parameters default values
  • If no argument is provided, the default is used
  • Default parameters must come after required ones
Default Example
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")  # Output: Hello, Alice!
greet("Bob", "Hi")  # Output: Hi, Bob!

ðŸ”§ Slide 7: Return Values

Getting Results from Functions

Returning Values

  • Use the return keyword to send a value back
  • Functions can return any type of value
  • Once return is executed, the function ends
Return Example
def add(a, b):
    return a + b

result = add(2, 3)
print(result)  # Output: 5

ðŸ”§ Slide 8: Multiple Return Values

Returning More Than One Value

Returning Multiple Values

  • Use a comma to return multiple values (as a tuple)
  • Unpack the result into multiple variables
Multiple Returns
def min_max(numbers):
    return min(numbers), max(numbers)

low, high = min_max([1, 2, 3, 4, 5])
print(low, high)  # Output: 1 5

ðŸ”§ Slide 9: Scope

Where Variables Live

Understanding Scope

  • Variables defined inside a function are local
  • Local variables can't be used outside the function
  • Global variables are defined outside all functions
  • Use global keyword to modify globals (not recommended)
Scope Example
def foo():
    x = 10  # Local variable
    print(x)

foo()
# print(x)  # Error: x is not defined

y = 5  # Global variable
def bar():
    print(y)

bar()  # Output: 5

ðŸ”§ Slide 10: Docstrings

Documenting Your Functions

What is a Docstring?

  • A docstring is a special string at the start of a function
  • It describes what the function does
  • Use triple quotes """ ... """
  • Helps others (and you) understand your code
Docstring Example
def square(x):
    """Returns the square of x."""
    return x * x

print(square(4))  # Output: 16
print(square.__doc__)  # Output: Returns the square of x.

ðŸ“‹ Slide 11: List Basics

Storing Multiple Values

What is a List?

  • A list is a collection of items in a specific order
  • Lists can store any type of data (numbers, strings, etc.)
  • Lists are defined with square brackets []
  • Items are separated by commas
List Example
# Creating a list
fruits = ["apple", "banana", "orange"]
print(fruits)

# Empty list
empty = []

# List with mixed types
mixed = [1, "hello", 3.14, True]
print(mixed)

ðŸ“‹ Slide 12: Accessing List Items

Getting Data from Lists

Accessing Elements

  • Use square brackets and an index to access items
  • Indexes start at 0 (first item)
  • Negative indexes count from the end
Access Example
fruits = ["apple", "banana", "orange"]
print(fruits[0])  # apple
print(fruits[2])  # orange
print(fruits[-1]) # orange

ðŸ“‹ Slide 13: Modifying Lists

Changing List Contents

Modifying List Items

  • Assign a new value to a specific index
  • Use append() to add items
  • Use remove() or pop() to delete items
Modify Example
numbers = [1, 2, 3]
numbers[0] = 10
print(numbers)  # [10, 2, 3]
numbers.append(4)
print(numbers)  # [10, 2, 3, 4]
numbers.remove(2)
print(numbers)  # [10, 3, 4]
numbers.pop()
print(numbers)  # [10, 3]

ðŸ“‹ Slide 14: List Methods

Built-in List Operations

Common List Methods

  • append(): Add item to end
  • insert(): Add item at position
  • remove(): Remove first occurrence
  • pop(): Remove and return item
  • sort(): Sort the list
  • reverse(): Reverse the list
  • len(): Get number of items
Methods Example
numbers = [3, 1, 4, 2]
numbers.sort()
print(numbers)  # [1, 2, 3, 4]
numbers.reverse()
print(numbers)  # [4, 3, 2, 1]
print(len(numbers))  # 4
numbers.insert(1, 99)
print(numbers)  # [4, 99, 3, 2, 1]

ðŸ“‹ Slide 15: Looping Through Lists

Processing List Items

Iterating Over Lists

  • Use a for loop to process each item
  • Use enumerate() to get index and value
  • List comprehensions for concise loops
Loop Example
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

for i, fruit in enumerate(fruits):
    print(i, fruit)

# List comprehension
lengths = [len(fruit) for fruit in fruits]
print(lengths)  # [5, 6, 6]

ðŸ“‹ Slide 16: Slicing Lists

Getting Parts of a List

List Slicing

  • Use list[start:end] to get a sublist
  • Start is inclusive, end is exclusive
  • Omit start or end for defaults
Slicing Example
numbers = [0, 1, 2, 3, 4, 5]
print(numbers[1:4])  # [1, 2, 3]
print(numbers[:3])   # [0, 1, 2]
print(numbers[3:])   # [3, 4, 5]
print(numbers[-2:])  # [4, 5]

ðŸ“‹ Slide 17: Copying Lists

Making List Copies

Copying Lists

  • Use list.copy() or slicing to copy a list
  • Direct assignment creates a reference, not a copy
Copy Example
numbers = [1, 2, 3]
copy1 = numbers.copy()
copy2 = numbers[:]

numbers[0] = 99
print(numbers)  # [99, 2, 3]
print(copy1)    # [1, 2, 3]
print(copy2)    # [1, 2, 3]

ðŸ“‹ Slide 18: Nested Lists

Lists Inside Lists

What are Nested Lists?

  • A list can contain other lists as elements
  • Useful for representing tables or grids
  • Access elements with double indexing
Nested Example
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]
print(matrix[0][1])  # 2
for row in matrix:
    print(row)

ðŸ“‹ Slide 19: List Comprehensions

Advanced List Creation

List Comprehensions

  • Concise way to create lists from other lists
  • Syntax: [expression for item in list]
  • Can include conditions: [x for x in numbers if x % 2 == 0]
Comprehension Example
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]
evens = [x for x in numbers if x % 2 == 0]
print(squares)  # [1, 4, 9, 16, 25]
print(evens)    # [2, 4]

ðŸ“‹ Slide 20: List Functions

Useful Built-in Functions

Built-in List Functions

  • len(list): Number of items
  • sum(list): Sum of items
  • min(list): Smallest item
  • max(list): Largest item
  • sorted(list): New sorted list
Functions Example
numbers = [3, 1, 4, 2]
print(len(numbers))    # 4
print(sum(numbers))    # 10
print(min(numbers))    # 1
print(max(numbers))    # 4
print(sorted(numbers)) # [1, 2, 3, 4]

ðŸ“‹ Slide 21: Filtering Lists

Selecting Items by Condition

Filtering with Loops and Comprehensions

  • Use a for loop to select items that meet a condition
  • List comprehensions make filtering concise
Filtering Example
numbers = [1, 2, 3, 4, 5, 6]
evens = []
for num in numbers:
    if num % 2 == 0:
        evens.append(num)
print(evens)  # [2, 4, 6]

# List comprehension
evens2 = [x for x in numbers if x % 2 == 0]
print(evens2)  # [2, 4, 6]

Working with Tables

2D Lists

  • Lists can contain other lists (rows and columns)
  • Useful for representing tables, grids, or matrices

ðŸ“‹ Slide 24: Common List Mistakes

Avoiding Pitfalls

Common Mistakes

  • Using = instead of == for comparison
  • Forgetting lists are zero-indexed
  • Modifying a list while looping over it
  • Confusing append() with extend()
Mistakes Example
# Mistake: = instead of ==
if item = "apple":  # Error!
    print("Found apple")

# Correct:
if item == "apple":
    print("Found apple")

# Mistake: Modifying list while looping
numbers = [1, 2, 3, 4]
for num in numbers:
    if num % 2 == 0:
        numbers.remove(num)
print(numbers)  # Unexpected result!

# Correct way:
numbers = [1, 2, 3, 4]
new_numbers = [num for num in numbers if num % 2 != 0]
print(new_numbers)  # [1, 3]
