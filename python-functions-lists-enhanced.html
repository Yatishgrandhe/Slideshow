Reusable Code and Data Collections

Welcome to Functions & Lists!

This course will teach you how to create reusable code with functions and work with collections of data using lists. These are fundamental concepts that will make your programs more powerful and organized.

What You'll Learn:

  • Function Basics: Defining and calling functions
  • Parameters: Passing data to functions
  • Return Values: Getting results from functions
  • List Creation: Making and manipulating lists
  • List Methods: Built-in list operations
  • List Comprehensions: Advanced list creation

Functions & Lists Examples
# Function definition
def greet(name):
    return f"Hello, {name}!"

# Function call
message = greet("Alice")
print(message)

# List creation
numbers = [1, 2, 3, 4, 5]
names = ["Alice", "Bob", "Charlie"]

# List operations
numbers.append(6)
names.remove("Bob")

# Function with list parameter
def calculate_average(numbers):
    if len(numbers) == 0:
        return 0
    return sum(numbers) / len(numbers)

# List comprehension
squares = [x**2 for x in range(5)]

# Function returning list
def get_even_numbers(max_num):
    return [x for x in range(max_num) if x % 2 == 0]
