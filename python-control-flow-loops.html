Performing Calculations in Python

Python Math Operators:

Python can perform all the mathematical operations you learned in school, plus some additional ones that are useful in programming.

Basic Arithmetic Operators:

  • Addition (+): Adds two numbers together
  • Subtraction (-): Subtracts the second number from the first
  • Multiplication (*): Multiplies two numbers
  • Division (/): Divides the first number by the second (gives decimal result)
  • Floor Division (//): Divides and rounds down to the nearest whole number
  • Modulo (%): Gives the remainder after division
  • Exponentiation (**): Raises a number to a power

Order of Operations:

Python follows the same math rules you learned in school:

  1. Parentheses: Do calculations inside () first
  2. Exponents: Calculate powers (**)
  3. Multiplication/Division: From left to right
  4. Addition/Subtraction: From left to right

Working with Variables:

  • Store calculation results in variables
  • Use variables in calculations
  • Update variables with new calculations
  • Combine different data types carefully
Math Operation Examples
# Basic arithmetic
a = 10
b = 3

print(a + b)   # 13 (addition)
print(a - b)   # 7 (subtraction)
print(a * b)   # 30 (multiplication)
print(a / b)   # 3.333... (division)
print(a // b)  # 3 (floor division)
print(a % b)   # 1 (remainder)
print(a ** b)  # 1000 (10 to the power of 3)

# Order of operations
result = 2 + 3 * 4
print(result)  # 14 (multiply first, then add)

# Using parentheses
result = (2 + 3) * 4
print(result)  # 20 (add first, then multiply)

# Working with variables
price = 19.99
quantity = 3
total = price * quantity
tax = total * 0.08
final_total = total + tax

print(f"Price: ${price}")
print(f"Quantity: {quantity}")
print(f"Subtotal: ${total}")
print(f"Tax: ${tax:.2f}")
print(f"Final Total: ${final_total:.2f}")

# Integer vs Float division
print(10 / 3)   # 3.333... (float result)
print(10 // 3)  # 3 (integer result)
print(10 % 3)   # 1 (remainder)

ðŸ”— Day 2: Strings & Concatenation

Working with Text in Python

What are Strings?

Strings are sequences of characters (letters, numbers, symbols) that represent text. They're one of the most commonly used data types in programming.

Creating Strings:

  • Single Quotes: 'Hello World'
  • Double Quotes: "Hello World"
  • Triple Quotes: """Multi-line strings"""
  • Escape Characters: Use \ for special characters

String Concatenation:

Concatenation means joining strings together. There are several ways to do this in Python:

  • + Operator: Joins strings with the + symbol
  • f-strings: Modern way to embed variables in strings
  • .format(): Older method for string formatting
  • % Operator: C-style string formatting

Why Use Different Methods?

  • f-strings: Most readable and recommended
  • + Operator: Simple for joining a few strings
  • .format(): Good for complex formatting
  • % Operator: Legacy code compatibility
String Examples
# Creating strings
first_name = "Jane"
last_name = "Doe"
age = 25

# String concatenation with +
full_name = first_name + " " + last_name
print(full_name)  # Jane Doe

# f-strings (recommended)
greeting = f"Hello, {first_name} {last_name}!"
print(greeting)  # Hello, Jane Doe!

# Using f-strings with expressions
message = f"{first_name} is {age} years old"
print(message)  # Jane is 25 years old

# .format() method
template = "Welcome, {} {}!"
formatted = template.format(first_name, last_name)
print(formatted)  # Welcome, Jane Doe!

# Multiple concatenations
part1 = "Python"
part2 = " is"
part3 = " awesome!"
result = part1 + part2 + part3
print(result)  # Python is awesome!

# Working with numbers in strings
price = 19.99
quantity = 3
total = price * quantity

# f-string with formatting
print(f"Price: ${price:.2f}")
print(f"Quantity: {quantity}")
print(f"Total: ${total:.2f}")

# Escape characters
print("She said \"Hello!\"")  # She said "Hello!"
print('It\'s a beautiful day')  # It's a beautiful day
print("Line 1\nLine 2")  # Multi-line output

ðŸ”§ Day 3: String Methods

Built-in Functions for Text Manipulation

What are String Methods?

String methods are built-in functions that help you manipulate and work with text. They make it easy to transform, search, and analyze strings.

Common String Methods:

  • .upper(): Converts all letters to uppercase
  • .lower(): Converts all letters to lowercase
  • .title(): Capitalizes the first letter of each word
  • .strip(): Removes whitespace from beginning and end
  • .replace(): Replaces one substring with another
  • .split(): Splits a string into a list of substrings
  • .join(): Joins a list of strings into one string
  • .find(): Finds the position of a substring
  • .count(): Counts how many times a substring appears

How to Use String Methods:

  • Syntax: string.method_name()
  • Chaining: You can use multiple methods together
  • Immutability: Original string doesn't change
  • Return Value: Methods return a new string

Practical Applications:

  • Cleaning user input (removing extra spaces)
  • Formatting text for display
  • Searching and replacing text
  • Parsing data from files or user input
String Method Examples
# Basic string methods
text = "  Hello World  "

# Case conversion
print(text.upper())      # "  HELLO WORLD  "
print(text.lower())      # "  hello world  "
print(text.title())      # "  Hello World  "

# Cleaning whitespace
print(text.strip())      # "Hello World"
print(text.lstrip())     # "Hello World  "
print(text.rstrip())     # "  Hello World"

# Replacing text
message = "Hello Python"
new_message = message.replace("Python", "World")
print(new_message)       # "Hello World"

# Splitting and joining
sentence = "Python is awesome"
words = sentence.split()  # ["Python", "is", "awesome"]
print(words)

# Joining words back together
new_sentence = " ".join(words)
print(new_sentence)      # "Python is awesome"

# Finding and counting
text = "Python programming is fun"
print(text.find("programming"))  # 7 (position)
print(text.count("m"))          # 2 (count of 'm')

# Chaining methods
user_input = "  HELLO WORLD  "
clean_input = user_input.strip().lower()
print(clean_input)       # "hello world"

# Practical example
name = input("Enter your name: ")
clean_name = name.strip().title()
print(f"Welcome, {clean_name}!")

# What you'll see:
# Enter your name:   john doe  
# Welcome, John Doe!

ðŸŽ¯ Day 4: Conditional Statements

Making Decisions in Your Code

What are Conditional Statements?

Conditional statements allow your program to make decisions based on certain conditions. They control which code runs and which doesn't.

Using if, elif, and else:

  • if: Checks if a condition is True
  • elif: Checks another condition if the first was False
  • else: Runs if all previous conditions were False
  • Indentation: Code inside conditions must be indented

Comparison Operators:

  • ==: Equal to
  • !=: Not equal to
  • <: Less than
  • >: Greater than
  • <=: Less than or equal to
  • >=: Greater than or equal to

Logical Operators:

  • and: Both conditions must be True
  • or: At least one condition must be True
  • not: Reverses the condition

Common Patterns:

  • Checking user input
  • Validating data
  • Making decisions in games
  • Handling different scenarios
Conditional Examples
# Basic if statement
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Using elif for multiple conditions
score = 85
if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
else:
    grade = "F"
print(f"Your grade is: {grade}")

# Comparison operators
x = 10
y = 5

if x > y:
    print("x is greater than y")
elif x < y:
    print("x is less than y")
else:
    print("x equals y")

# Logical operators
username = "admin"
password = "12345"

if username == "admin" and password == "12345":
    print("Login successful")
else:
    print("Login failed")

# Checking user input
user_input = input("Enter a number: ")
number = int(user_input)

if number > 0:
    print("Positive number")
elif number < 0:
    print("Negative number")
else:
    print("Zero")

# Nested conditions
age = 25
has_license = True

if age >= 16:
    if has_license:
        print("You can drive")
    else:
        print("You need a license")
else:
    print("Too young to drive")

# Using 'in' operator
fruits = ["apple", "banana", "orange"]
user_fruit = input("Enter a fruit: ")

if user_fruit in fruits:
    print("That fruit is in our list!")
else:
    print("Fruit not found")

ðŸ”„ Day 5: Loops Introduction

Repeating Code with Loops

What are Loops?

Loops allow you to repeat code multiple times without writing it over and over. They're essential for processing data and automating tasks.

Types of Loops:

  • for loops: Repeat a specific number of times
  • while loops: Repeat while a condition is True
  • Nested loops: Loops inside other loops

For Loops with range():

  • range(n): Numbers from 0 to n-1
  • range(start, end): Numbers from start to end-1
  • range(start, end, step): Numbers with step size
  • Iterating over lists: Loop through each item

While Loops:

  • Condition-based: Repeat while condition is True
  • Manual control: You control when to stop
  • Infinite loops: Be careful to avoid these
  • Break statement: Exit loop early

Loop Control:

  • break: Exit the loop immediately
  • continue: Skip to next iteration
  • else clause: Run when loop finishes normally
Loop Examples
# For loop with range
for i in range(5):
    print(f"Count: {i}")

# Output: Count: 0, 1, 2, 3, 4

# For loop with start and end
for i in range(1, 6):
    print(f"Number: {i}")

# Output: Number: 1, 2, 3, 4, 5

# For loop with step
for i in range(0, 10, 2):
    print(f"Even: {i}")

# Output: Even: 0, 2, 4, 6, 8

# Iterating over a list
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(f"I like {fruit}")

# While loop
count = 0
while count < 5:
    print(f"Count: {count}")
    count += 1

# While loop with break
number = 0
while True:
    if number >= 5:
        break
    print(f"Number: {number}")
    number += 1

# Loop with continue
for i in range(10):
    if i % 2 == 0:  # Skip even numbers
        continue
    print(f"Odd: {i}")

# Nested loops
for i in range(3):
    for j in range(3):
        print(f"({i}, {j})", end=" ")
    print()  # New line after each row

# Loop with else
for i in range(5):
    print(i)
else:
    print("Loop completed successfully!")
