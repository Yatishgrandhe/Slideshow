Making Decisions and Repeating Actions

Welcome to Control Flow & Loops!

This course will teach you how to make your programs make decisions and repeat actions. You'll learn about conditionals, loops, and how to control the flow of your programs.

What You'll Learn:

  • Conditional Statements: if, elif, else
  • Comparison Operators: ==, !=, <, >, <=, >=
  • Logical Operators: and, or, not
  • For Loops: Iterating over sequences
  • While Loops: Repeating while condition is true
  • Control Statements: break, continue, pass

Course Structure:

  • Slides 1-5: Introduction and Basics
  • Slides 6-10: Conditional Statements
  • Slides 11-15: Comparison and Logical Operators
  • Slides 16-20: For Loops
  • Slides 21-25: While Loops and Advanced Topics
Control Flow Examples
# Conditional statement
age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# For loop
for i in range(5):
    print(f"Count: {i}")

# While loop
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# Logical operators
name = "Alice"
age = 25
if name == "Alice" and age >= 18:
    print("Welcome, Alice!")
