Your First Programming Language

What is Python?

Python is a high-level, interpreted programming language designed to be readable and simple, making it perfect for beginners.

Why Python?

  • Easy to Learn: Python reads almost like English
  • Versatile: Used in web development, data science, automation, and AI
  • Popular: One of the most in-demand programming languages
  • Beginner-Friendly: Perfect starting point for your coding journey

Where is Python Used?

  • Web Development: Building websites and web applications
  • Data Science: Analyzing data and creating visualizations
  • Automation: Automating repetitive tasks
  • Artificial Intelligence: Machine learning and AI applications
  • Game Development: Creating games and interactive applications
Python Code Example
# This is how simple Python is!
print("Hello, World!")

# Variables are easy to understand

age = 25
print(f"{name} is {age} years old")

# Simple math operations

What is "Hello World"?

Why Start with Hello World?

üí¨ Comments in Python

Explaining Your Code

What are Comments?

Comments are notes you write in your code to explain what the code does. They help you and others understand your code better.

Why Use Comments?

  • Documentation: Explain what your code does
  • Memory Aid: Remind yourself of your logic later
  • Teamwork: Help others understand your code
  • Debugging: Temporarily disable code to test
  • Learning: Great for taking notes while learning

How to Write Comments:

  • Single Line: Use # at the beginning of the line
  • Inline: Add # after code on the same line
  • Multi-line: Use triple quotes for longer explanations
  • Clear Language: Write in plain English

Best Practices:

  • Explain the "why" not just the "what"
  • Keep comments up to date with your code
  • Use comments to break up long sections
  • Don't over-comment obvious code
Comment Examples
# This is a single-line comment
print("Hello")  # This is an inline comment

# Calculate the area of a circle
radius = 5
area = 3.14 * radius * radius
print(f"Area: {area}")

# Multi-line comment using triple quotes
"""
This is a longer comment
that can span multiple lines.
Useful for explaining complex code.
"""

# Good comment example
age = 25
# Check if person is old enough to vote
if age >= 18:
    print("Can vote!")
else:
    print("Too young")

# Bad comment example (too obvious)
name = "Alice"  # Set name to Alice
print(name)     # Print the name

üì¶ Variables and Types

Storing Information

What are Variables?

Variables are like labeled boxes where you store information. They can hold different types of data that your program can use.

How Variables Work:

  • Assignment: Use = to put data in a variable
  • Naming: Choose descriptive names (use letters, numbers, underscore)
  • Dynamic Typing: Python figures out the data type automatically
  • Reusable: Use the variable name to access the stored data

Python Data Types:

  • String (str): Text data - "Hello", "Python"
  • Integer (int): Whole numbers - 5, 100, -3
  • Float: Decimal numbers - 3.14, 2.5, -1.0
  • Boolean (bool): True or False values
  • List: Collection of items - [1, 2, 3]

Variable Naming Rules:

  • Start with a letter or underscore
  • Can contain letters, numbers, and underscores
  • Case sensitive (name ‚â† Name)
  • Use descriptive names (user_age, not x)
  • Avoid Python keywords (print, if, for, etc.)

üîç Type Checking & Casting

Understanding and Converting Data Types

What is Type Checking?

Type checking helps you understand what kind of data a variable contains. This is useful for debugging and ensuring your code works correctly.

Using type() Function:

  • Check Type: Use type(variable_name) to see the data type
  • Debugging: Helps identify why calculations might not work
  • Learning: Great way to understand what Python is doing
  • Validation: Ensure you're working with the right data type

What is Type Casting?

Type casting (or conversion) is the process of changing one data type to another. This is often necessary when working with user input or combining different types of data.

Common Conversion Functions:

  • int(): Convert to integer (whole number)
  • float(): Convert to decimal number
  • str(): Convert to text string
  • bool(): Convert to True/False

When to Use Type Casting:

  • User input always comes as text (string)
  • Mathematical calculations need numbers
  • Displaying numbers as text
  • Combining different data types
Type Checking & Casting Examples
# Type checking examples
age = 25
name = "Alice"
height = 5.6
is_student = True

print(type(age))        # 
print(type(name))       # 
print(type(height))     # 
print(type(is_student)) # 

# Type casting examples
# String to Integer
text_number = "42"
number = int(text_number)
print(number)  # 42
print(type(number))  # 

# Integer to String
age = 25
age_text = str(age)
print(age_text)  # "25"
print(type(age_text))  # 

# String to Float
decimal_text = "3.14"
pi = float(decimal_text)
print(pi)  # 3.14

# Float to Integer
price = 19.99
whole_price = int(price)
print(whole_price)  # 19 (truncates decimal)

# Boolean conversion
print(bool(1))    # True
print(bool(0))    # False
print(bool(""))   # False
print(bool("Hi")) # True

üì• Getting Input

Making Your Programs Interactive

What is User Input?

User input allows your program to get information from the person using it. This makes your programs interactive and dynamic.

Using the input() Function:

  • Basic Syntax: input("Your message here")
  • Prompt Message: The text in quotes tells the user what to enter
  • Always Returns String: Input is always treated as text initially
  • Program Pauses: Your program waits for the user to type something

Important Things to Remember:

  • String Output: input() always returns a string, even if the user types a number
  • Type Conversion: You often need to convert input to the right type
  • Error Handling: Users might enter unexpected data
  • Clear Prompts: Tell users exactly what you want them to enter

Common Input Patterns:

  • Getting names and text information
  • Collecting numbers for calculations
  • Asking for yes/no answers
  • Building interactive games and tools

üßÆ Basic Math Operations

Performing Calculations in Python

Python Math Operators:

Python can perform all the mathematical operations you learned in school, plus some additional ones that are useful in programming.

Basic Arithmetic Operators:

  • Addition (+): Adds two numbers together
  • Subtraction (-): Subtracts the second number from the first
  • Multiplication (*): Multiplies two numbers
  • Division (/): Divides the first number by the second (gives decimal result)
  • Floor Division (//): Divides and rounds down to the nearest whole number
  • Modulo (%): Gives the remainder after division
  • Exponentiation (**): Raises a number to a power

Order of Operations:

Python follows the same math rules you learned in school:

  1. Parentheses: Do calculations inside () first
  2. Exponents: Calculate powers (**)
  3. Multiplication/Division: From left to right
  4. Addition/Subtraction: From left to right

Working with Variables:

  • Store calculation results in variables
  • Use variables in calculations
  • Update variables with new calculations
  • Combine different data types carefully
Math Operation Examples
# Basic arithmetic
a = 10
b = 3

print(a + b)   # 13 (addition)
print(a - b)   # 7 (subtraction)
print(a * b)   # 30 (multiplication)
print(a / b)   # 3.333... (division)
print(a // b)  # 3 (floor division)
print(a % b)   # 1 (remainder)
print(a ** b)  # 1000 (10 to the power of 3)

# Order of operations
result = 2 + 3 * 4
print(result)  # 14 (multiply first, then add)

# Using parentheses
result = (2 + 3) * 4
print(result)  # 20 (add first, then multiply)

# Working with variables
price = 19.99
quantity = 3
total = price * quantity
tax = total * 0.08
final_total = total + tax

print(f"Price: ${price}")
print(f"Quantity: {quantity}")
print(f"Subtotal: ${total}")
print(f"Tax: ${tax:.2f}")
print(f"Final Total: ${final_total:.2f}")

# Integer vs Float division
print(10 / 3)   # 3.333... (float result)
print(10 // 3)  # 3 (integer result)
print(10 % 3)   # 1 (remainder)

üîó Strings & Concatenation

Working with Text in Python

What are Strings?

Strings are sequences of characters (letters, numbers, symbols) that represent text. They're one of the most commonly used data types in programming.

Creating Strings:

  • Single Quotes: 'Hello World'
  • Double Quotes: "Hello World"
  • Triple Quotes: """Multi-line strings"""
  • Escape Characters: Use \ for special characters

String Concatenation:

Concatenation means joining strings together. There are several ways to do this in Python:

  • + Operator: Joins strings with the + symbol
  • f-strings: Modern way to embed variables in strings
  • .format(): Older method for string formatting
  • % Operator: C-style string formatting

Why Use Different Methods?

  • f-strings: Most readable and recommended
  • + Operator: Simple for joining a few strings
  • .format(): Good for complex formatting
  • % Operator: Legacy code compatibility
String Examples
# Creating strings
first_name = "Jane"
last_name = "Doe"
age = 25

# String concatenation with +
full_name = first_name + " " + last_name
print(full_name)  # Jane Doe

# f-strings (recommended)
greeting = f"Hello, {first_name} {last_name}!"
print(greeting)  # Hello, Jane Doe!

# Using f-strings with expressions
message = f"{first_name} is {age} years old"
print(message)  # Jane is 25 years old

# .format() method
template = "Welcome, {} {}!"
formatted = template.format(first_name, last_name)
print(formatted)  # Welcome, Jane Doe!

# Multiple concatenations
part1 = "Python"
part2 = " is"
part3 = " awesome!"
result = part1 + part2 + part3
print(result)  # Python is awesome!

# Working with numbers in strings
price = 19.99
quantity = 3
total = price * quantity

# f-string with formatting
print(f"Price: ${price:.2f}")
print(f"Quantity: {quantity}")
print(f"Total: ${total:.2f}")

# Escape characters
print("She said \"Hello!\"")  # She said "Hello!"
print('It\'s a beautiful day')  # It's a beautiful day
print("Line 1\nLine 2")  # Multi-line output

üîß String Methods

Built-in Functions for Text Manipulation

What are String Methods?

String methods are built-in functions that help you manipulate and work with text. They make it easy to transform, search, and analyze strings.

Common String Methods:

  • .upper(): Converts all letters to uppercase
  • .lower(): Converts all letters to lowercase
  • .title(): Capitalizes the first letter of each word
  • .strip(): Removes whitespace from beginning and end
  • .replace(): Replaces one substring with another
  • .split(): Splits a string into a list of substrings
  • .join(): Joins a list of strings into one string
  • .find(): Finds the position of a substring
  • .count(): Counts how many times a substring appears

How to Use String Methods:

  • Syntax: string.method_name()
  • Chaining: You can use multiple methods together
  • Immutability: Original string doesn't change
  • Return Value: Methods return a new string

Practical Applications:

  • Cleaning user input (removing extra spaces)
  • Formatting text for display
  • Searching and replacing text
  • Parsing data from files or user input
String Method Examples
# Basic string methods
text = "  Hello World  "

# Case conversion
print(text.upper())      # "  HELLO WORLD  "
print(text.lower())      # "  hello world  "
print(text.title())      # "  Hello World  "

# Cleaning whitespace
print(text.strip())      # "Hello World"
print(text.lstrip())     # "Hello World  "
print(text.rstrip())     # "  Hello World"

# Replacing text
message = "Hello Python"
new_message = message.replace("Python", "World")
print(new_message)       # "Hello World"

# Splitting and joining
sentence = "Python is awesome"
words = sentence.split()  # ["Python", "is", "awesome"]
print(words)

# Joining words back together
new_sentence = " ".join(words)
print(new_sentence)      # "Python is awesome"

# Finding and counting
text = "Python programming is fun"
print(text.find("programming"))  # 7 (position)
print(text.count("m"))          # 2 (count of 'm')

# Chaining methods
user_input = "  HELLO WORLD  "
clean_input = user_input.strip().lower()
print(clean_input)       # "hello world"

# Practical example
name = input("Enter your name: ")
clean_name = name.strip().title()
print(f"Welcome, {clean_name}!")

# What you'll see:
# Enter your name:   john doe  
# Welcome, John Doe!
